## Описание

Учебный проект в стиле Trello:
- 3 фиксированные колонки;
- добавление карточек;
- удаление карточек по крестику на hover;
- drag-and-drop внутри колонки и между колонками;
- сохранение состояния в LocalStorage;
- полная сборка через Webpack.

## Скрипты

- `yarn start` — запуск dev-сервера;
- `yarn build` — production-сборка;
- `yarn lint` — ESLint;
- `yarn test` — Jest.

## Технологии

- JavaScript (ES6+)
- Webpack
- Babel
- ESLint
- Jest
- GitHub Actions + GitHub Pages
